# Diver Quartermaster

| Diver Quartermaster | |
|---|---|
| name | Diver Quartermaster |
| location | Aretal sea (random sea event) |

## Description

The Diver Quartermaster is the leader of an expedition to retrieve a crate found in [Fort Merit](Fort_Merit.md).

Returning the crate to the Diver Quartermaster rewards you with an [Armorer's Needle](<Armorer's_Needle.md>), a [Smith's Alloy](<Smith's_Alloy.md>), and a [Knowledge](Knowledge.md) point.
