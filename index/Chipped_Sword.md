# Chipped Sword

| Chipped sword | |
|---|---|

## Description

The Chipped Sword is a [Quest](Quests.md) item(NPC) found at the [Isle of Vigils](Isle_of_Vigils.md). It presumably belonged to [Nanji](Nanji.md)'s Brother [Ilji](Ilji.md), but he seems to have departed with the sword as it is in a very bad and rusty condition, as well as [Ilji](Ilji.md) himself being nowhere to be found.
