# Inn

## Description

Inns are locations within every populated faction base that allow you to change a player's spawn location whenever they interact with an [Innkeeper](Innkeep.md).

The following is a list of where Inns can be found:

-   [Etris](Etris.md)
-   [Isle of Vigils](Isle_of_Vigils.md)
-   [Miner's Landing](<Miner's_Landing.md>)
-   [Meteor Isle](Meteor_Isle.md)
-   [Greathive Aratel](Greathive_Aratel.md)
-   [Port Merit](Port_Merit.md)
-   [Summer Isle](Summer_Isle.md)
