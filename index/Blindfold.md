# Blindfold

Template:EquipmentStats

## Description

The Blindfold is an equipment piece you can find in both [Erisia's](Erisia.md). Equiping this item will give you the [Blinded](Talents.md) talent, which blinds you (not fully blind, just makes everything hard to see). The Blindfold is required for the [Blindseer](Oaths.md) oath.

## Notes

The Blindfold, Crimson Blindfold and Black Blindfold are all the same item with different appearances.
