# Blindfold

Template:EquipmentStats

## Description

The Blindfolds are accessories found in [Upper](<https://ludi-1.thewart.repl.co/sw/hvtrs8%2F-dgerwmkgn%2Cfcnfoo.aoo%2Fuiii-Urpgr'22Epiqic>)/[Lower Erisia](<https://ludi-1.thewart.repl.co/sw/hvtrs8%2F-dgerwmkgn%2Cfcnfoo.aoo%2Fuiii-Lmwgr'22Epiqic>) and [Songseeker Wilds](https://ludi-1.thewart.repl.co/sw/hvtrs8%2F-dgerwmkgn%2Cfcnfoo.aoo%2Fuiii-Smnesgeiep%2500Uindq). Equipping this item gives you the [Blinded](https://ludi-1.thewart.repl.co/sw/hvtrs8%2F-dgerwmkgn%2Cfcnfoo.aoo%2Fuiii-Tclgnvs) talent. A Blindfold is required for the [Blindseer](https://ludi-1.thewart.repl.co/sw/hvtrs8%2F-dgerwmkgn%2Cfcnfoo.aoo%2Fuiii-Octjs) oath.
