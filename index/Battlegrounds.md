# Battlegrounds

Deepwoken Battlegrounds is the April Fools 2025 joke for Deepwoken. It uses the [Battle Royale](Battle_Royale.md) map for its map and has all the weapon, shrine, and chest spawns.

There is also a tool given to everyone called "Ether Fly" which allows the user to fly around with zero cost.

Players also get every mantra in the game and 50 in every stat.
