# Scope

## Description

Scope is a [Vesperian](Vesperian.md) NPC in the [Summer Isle](Summer_Isle.md).

He is collecting explosive boxes found in events all over the Summer Isle. For those who turn an explosive box, he will reward them with a chest and positive [Summer](The_Summer_Company.md) [Company](The_Summer_Company.md) reputation.

This is a good way to earn some easy reputation, much like the [Etris Wood Quest](Korin.md). This is great, because occasionally [Summer Company](The_Summer_Company.md) events will attack neutral players, but not any with more than that.
