# Eyeglasses

Template:EquipmentStats

## Description

The Eyeglasses are an Equipment item you can obtain in Minityrsa and Songseeker Chests. They provide a random stat buff when worn.

## Notes

The Eyeglasses and Polarized Eyeglasses are the exact same item, only with different appearances.
