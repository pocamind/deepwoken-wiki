# Ziplines

| Ziplines | |
|---|---|

## Description

Ziplines are stationary pulleys used to travel across great distances within a location.

## Usage

Ziplines can be used by pressing Space while standing near one, causing the user to descend rapidly towards their destination.

While descending, the user can press Space to leap forward and gain a considerable distance.
