# Chum

| Chum | |
|---|---|
| value | 10 |
| weight | 0.25 |
| type | * Buyable, Crafting with Fishmeat. |
| bankable | Yes |
| droppable | Yes |
| pass down | No |

You can use **Chum** as bait when [Fishing](Fishing.md). Fish meat can also be used as well, however this can be a better option.

You can also use 3 chum to craft all three types of Monster Bait.

Recipe - _2 Fish Meat_

Can be bought at the [Etris Docks](Etris_Docks.md) and directly from [Mortus](Mortus.md) near [Lower Erisia](Lower_Erisia.md) for.

Chum is also obtainable from Crustacean meat, by killing a [Crustaceous Rex](Crustaceous_Rex.md) and the [Crabs](Crab.md) in the Diluvian Mechanism.
