# Conquest

## Description

Conquest is an upcoming gamemode in which multiple teams take battle against eachother in return for end-game exclusive content, as well as a casual mode that you enter with a fresh character like Battle Royale, which boasts no rewards.

The Speculated Conquest Rewards are:

-   [Bloodsworn Effigy](Bloodsworn_Effigy.md)
-   [Oath: Saintsworn](Oath%253A_Saintsworn.md)

Conquest is set to release within the month of January 2023.Template:Areainfobox
