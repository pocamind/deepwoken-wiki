# Windmill Ring

**Windmill Ring** is an [Equipment](Equipment.md) item in _Deepwoken_.

## Description

Windmill Ring grants the user +2% Ice Armor, +2% Wind Armor, and one Uncommon Pip upon being equipped. It also causes the user to regenerate ether while taking damage from [ice](Frostdraw.md) or [wind](Galebreathe.md) mantras. The user must be at Power 3 to equip the ring.
