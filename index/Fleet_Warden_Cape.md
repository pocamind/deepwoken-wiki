# Fleet Warden Cape

“ THE ONE PIECE IS REAL! ”

― Average player when they find the Fleet Warden Cape for the first time.

  

## Description

Template:EquipmentStats The Fleet Warden Cape can be obtained from [fishing](Fishing.md) and [Summer Isle](Summer_Isle.md) chests. When equipped, you gain various buffs. Requires Power 15 to wear it.

It has 6 variants which are: Emerald, Rose, Scarlet, Shade, Silver & Onyx.
