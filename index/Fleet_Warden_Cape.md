# Fleet Warden Cape

_If you wish to see useful information on all Equipment, Go to [**Equipment**](Equipment.md)_

“ THE ONE PIECE IS REAL! ”

― Average player when they find the Fleet Warden Cape for the first time.

## Description

Template:EquipmentStats The Fleet Warden Cape can be obtained from [fishing](Fishing.md) and [Summer Isle](Summer_Isle.md) chests. When equipped, you gain +3% Elemental Armor and 1 Common [pip](Equipment.md). Requires Power 13 to wear it.

It has 6 variants which are: Emerald, Rose, Scarlet, Shade, Silver & Onyx.
