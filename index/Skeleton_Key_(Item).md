## Description

The Skeleton Key is an item obtained from either [Vow of Iron](Vow_of_Iron.md)'s Mystery Box or [Strange Merchant](Strange_Merchant.md)'s Magic Coffin.

It is intended to instantly open any chest without the need for antiquariums or chest masters.

Currently bugged and does absolutely nothing.

## Trivia

-   A skeleton key is a key which can unlock any lock.
