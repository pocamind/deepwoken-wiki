## Description

The Manor Key is a [Quest](Quests.md) item given to you by [The Lord Regent](The_Lord_Regent.md) in "The Lord Regent's quest" questline. The Manor key glows when it is able to be used to unlock [Manor Gate](<Duke_Erisia's_Manor.md>) and goes dark when the gate has already been opened.
