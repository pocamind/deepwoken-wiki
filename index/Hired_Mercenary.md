# Hired Mercenary

## Description

Template:ArmorInfobox **Hired Mercenary** is a Starter [Outfit](Outfits.md) in _Deepwoken_.

### _Requirements_

This Outfit costs and requires five (5) Cloth and Fiber, and one (1) piece of Iron.

This Outfit has no [Power](Power.md) or [Attribute](Attributes.md) requirement.

### _Armor Stats_

This Outfit has 1000 durability grants the user +15% Physical resistance and +10% Elemental resistance.

## Location

This Outfit's schematic can be found in:

-   [Lower Erisia](Lower_Erisia.md)

## Trivia/Notes

-   Oddly enough, given the name, you can see Karliah wearing this Outfit after completing [her quest](<https://deepwoken.fandom.com/wiki/Etris_(Starting_Island)?so=search#Diver_Escort_Quest>).
