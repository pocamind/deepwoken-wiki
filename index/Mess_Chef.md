# Mess Chef

| Mess Chef | |
|---|---|
| aspect | Randomized |
| location | Guild Base Mess Hall |

## Description

The **Mess Chef** is an NPC located inside of [Guild Bases](Guild_Bases.md). They can be hired for 2000 to prepare foods available at any time.

## Shop

-   [Mushroom Bisque](Mushroom_Bisque.md) for 25.
-   [Cheese](Cheese.md), [Fish Stew](Fish_Stew.md), [Mushroom Omelette](Mushroom_Omelette.md), and [Sushi](Sushi.md) for 20.
