## Description

The Odd Tentacle is a rare item dropped by [Nautilodaunts](Nautilodaunt.md) which can be found in [The Depths](The_Depths.md), [Starswept Valley](The_Starswept_Valley.md), [The Diluvian Mechanism](The_Diluvian_Mechanism.md), and from [Mimic Chests](Chests_&_Sacks.md). A rare drop, [Nautilodaunt Beak](Nautilodaunt_Beak.md), can be used to obtain [Abyssal Ridge](Monster_Mantras.md).
