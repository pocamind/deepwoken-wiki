# Celtorian Guard

Template:EnemyInfobox1.0

## Description

A Celtor Guard is an NPC that can only be found in [The Depths](The_Depths.md). They are found in and around Celtor. Talking to them will prompt the following dialogue (Needs more info):

"\*They shuffle strangely, as if righting themself.\*

"What is this place?"

They wear Barrel Helm and a Duelist's Attire outfit.

## Moveset

Their arsenal consists of either a [Messer](Messer.md) or [Trident Spear](Trident_Spear.md) (Needs more info):
