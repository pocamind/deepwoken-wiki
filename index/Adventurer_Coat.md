## Description

The Adventurer Coat is an animated equipment item in the arms category. Equipping it grants you 2 Pips. It can only be found in chests from [Duke Erisia](Duke_Ishamon_Erisia.md). It has 3 variants lacking pictures which are; White, Brown, and Green.
