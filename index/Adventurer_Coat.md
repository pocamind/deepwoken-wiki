# Adventurer Coat

Template:EquipmentStats

## Description

The Adventurer Coat is an armor piece part of the Arms Catagory. Equipping it grants you randomized stats based on the Pips you get. The Adventurer Coat can only be found in [Duke Erisia](Duke_Ishamon_Erisia.md)'s Chests
