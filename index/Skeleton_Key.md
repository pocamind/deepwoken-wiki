# Skeleton Key

| Skeleton Key | |

| appearance | A grey skeletal wall, with hands embodying the bottom and the top being a symbol. |

“ Summon skeletal barriers that trap opponents and launch them across the battlefield. ”

― - Skeleton Key Description

  

## Description

Skeleton Key is a [Resonance](Resonance.md) that players can obtain and is uncommon to roll.

### Appearance

When summoned, a grey wall will manifest, with the bottom part of it resembling skeleton hands and the top embodying a symbol.

If corrupted. the walls become a dark blue/violet.

## Usage

Pressing C will activate the **Skeleton Key** Resonance.

Skeleton key will summon a wall, up to three if not corrupted under a set duration. If touched the wall will push anyone indiscriminately, ally or not. However if they press C again, the wall will automatically be pushed.

If Skeleton Key is corrupted, you could summon up to five walls with a lesser duration.

### Resonance Point Usage

when obtained, Skeleton key will spend its points on the following stats:

-   **Wall type:** Decides the type of wall that will be used.
    -   Normal doesn't deal dmg on push
    -   Spiked deals dmg on push

-   **Wall Amount:** How many walls you can spawn. (not guaranteed 5 because you need the correct point usage)
-   **Duration:** How fast it takes to spawn a wall.
-   **Range (?):** How far a wall can push.
-   **Wall type:** Decides the type of wall that will be used.

## Trivia

-   Based off of the dev spec ability under the same name used by Zartania, rebalanced and reworked for public use.

-   Keep in mind that a corrupted Skeleton Key does not guarantee five walls, as the number is still based on the resonance point usage.
