## Description

**YUM YUM BAR** is a [Food](Food.md) item. It is exclusively obtained from the "Food" Machine in [The Hall of Shattered Wills](The_Hall_of_Shattered_Wills.md).

It is a type of **Meat/Plant/Yum**.

### Effect

When consumed, it replenishes 50% Stomach and 50% Water.
