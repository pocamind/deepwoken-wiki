# Officers Note

The Officers Note is an item that had a chance to drop from bandits in Lower Erisia, though it could of have been dropped by other enemies as well.

-   Allows for guards to address player by name
-   Gives access to the palace with sufficent rep ( eventually allows for acess to "Lord Regent" quest)
-   Consumable used to increase positive Etrea reputation
-   This item was only out for the first few weeks of Deepwokens release.  
    **This item has since been removed, or is unobtainable as of now (6-25-2022).**
