# Resonance Door

| Resonance Door | |
|---|---|
| usage | Access special areas |

## Description

A **Resonance Door** is a special door that require a [Resonance](Resonance.md) to open. They are found only within [The Depths](The_Depths.md), locking away certain areas or creating shortcuts that save the player's time.
