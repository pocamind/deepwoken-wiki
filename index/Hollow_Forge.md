# Hollow Forge

## Overview

Hollow Forge at minor landing

The Hollow forge is an interactable furnace that can only be used by those with Ignition Union Origin. It allows the conversion of Pluripotent Alloys into Smith's Alloys.

  

## Uses

Used to turn 3 [Pluripotent Alloy's](Pluripotent_Alloy.md) into 1 [Smith's Alloy](<Smith's_Alloy.md>).

This can be done indefinitely as long as you own enough Pluripotent Alloys.
