# Hollow Forge

## Hollow Forge

Hollow Forge at miner landing

The Hollow forge is an interactable furnace that can only be used by those with igniton union origin

  

  

  

## Uses

Used to turn 3 [Pluripotent Alloy's](Pluripotent_Alloy.md) into 1 [Smith's Alloy](<Smith's_Alloy.md>)

can only be done if you have [Ignition union origin](Origins.md)

  

## Trivia

was added on 4/10/24 to [Miner's Landing](<Miner's_Landing.md>) for the recently added [ignition union origin](Origins.md)

can be used infinite times only requiring 3 [Pluripotent Alloy's](Pluripotent_Alloy.md)
