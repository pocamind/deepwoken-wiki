# Armourer Kev

| Armourer Kev | |
|---|---|
| name | Kev |
| aspect | Adret |
| location | Fort Merit |

## Description

Armourer Kev is a [Adret](Adret.md) NPC, he is the armorer for [The Central Authority](The_Central_Authority.md) and sells a variety of [Weapons](Weapons.md).

## Location

Armourer Kev is located inside the bottom level of the fortress at [Fort Merit](Fort_Merit.md).
