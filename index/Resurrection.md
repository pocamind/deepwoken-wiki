# Resurrection

## Description

**Resurrection** is a [Resonance](Resonance.md) or "bell" that players can obtain, and is possibly the rarest to roll.

### _Appearance_

When used, the user's bell will emit gold particles.

## Usage

Pressing C will activate the **Resurrection** Resonance.

**Resurrection** can be used to make a ko'd friend up back on there feet again. It has a few second windup before you use your hands like a defibrillator and zap them awake. (I do not know if this only works on guild members or not)

## Trivia/Notes

-   Commonly known as the "placeholder bell." This bell will have a use later on.
