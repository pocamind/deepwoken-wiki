# Heavy Hands Ring

Heavy Hands Ring is an [Equipment](Equipment.md) item under the "Rings" category.

## Description

Heavy Hands Ring grants the user +30% posture damage and -10% swing speed on "M1" attacks, and one Uncommon Pip upon being equipped.
