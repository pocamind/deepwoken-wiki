# Tideknight

## Description

Tideknights can only found within the later waves of [The Diluvian Mechanism](The_Diluvian_Mechanism.md) in large numbers. Template:EnemyInfobox1.0 They wear the [Celtorian Tideknight](Celtorian_Tideknight.md) armor, hence the name, and use a wide array of legendary weapons.
