# Lifevents

| Lifevents | |
|---|---|

## Description

Lifevents, commonly referred to as Jump Pads, are stationary structures that are used to provide upward boosts. They have a geyser-like appearance, and emits a green mist moving up into the air. They are mainly found within the [Starswept Valley](The_Starswept_Valley.md), and in parts of the [Battle Royale](Battle_Royale.md) map.

## Usage

A lifevent can be used simply by pressing Space directly below one. If the user is lifted through this, they will not take any fall damage.

While falling, a user can land below a jump pad and gain an upward boost.
