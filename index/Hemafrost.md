# Hemafrost

“ A glowing sphere that emits ice energy. ”

― Description (I think)

The Hemafrost in the mantra shop at Isle of Vigils

## **Description**

_"A glowing sphere that emits ice energy."_

Trains the [Frostdraw attunement](Frostdraw.md). It can be used 200 times. Upon activation, the character is slowed and surrounded by a frosty glow. It can be used at [power](Power.md) level 2+ if your character is attuned to [frostdraw](Frostdraw.md) (has frostdraw in discovered elements followed by a number)

_Purchased at any [Mantra](Mantras.md) Shop - **25 [Notes](Notes.md)**_

## Trivia

-   Every attunement has a training item.
-   Can break if overused.
-   You can use this at power 1 easily by assassinating the Navaen Nomads and using the the Frigid Prisms to obtain the Frostdrawers talent via the trainer and buying Hemafrost.
