## Description

**Sauteed Scallops** is a craftable [Food](Food.md) item.

It is a type of **Meat**, meaning it cannot be eaten with the [Vegetarian](Boons_and_Flaws.md#vegetarian) Flaw.

### Effect

When consumed, it replenishes 25% Stomach and 5% Water.

### Recipe
