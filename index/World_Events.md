# World Events

## Overview

World events are game wide events that occur in every server once every real life hour (exactly at XX:00). Each [Luminants](Luminants.md) will have their own world events and every server of that luminant will have the same event.

World events are focused on uniting the players towards accomplishing a single goal and awarding them individually.
