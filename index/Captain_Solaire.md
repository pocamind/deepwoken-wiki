# Captain Solaire

Template:EnemyInfobox1.0

## Description

Captain Solaire is an NPC found in [Merit Detainment Core](Merit_Detainment_Core.md), a dangerous enemy which guards the important parts.

Their appearance consists of an [Authority Helmet](Authority_Helm.md), an [Authority Captain Uniform](Authority_Captain_Uniform.md) and an [Authority Commander Coat](Authority_Commander_Coat.md), albeit missing the sleeves.

## Moveset

Their arsenal consists of a warden ceremonial along with Thundercall~, Flamecharm and Frostdraw attunements.~ this one only uses thunder (over 15 dungeons)
