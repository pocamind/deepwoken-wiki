## Description

Unstable is an **unobtainable** RNG-based [enchantment](Enchantments.md) that can proc random enchantments.

## Effect

Unstable has a 10% chance on hit to proc any other enchantment in the game.

This is not a 10% chance to proc **a** enchantment, but rather each enchantment has a 10% chance to proc **individually** of all others, potentially allowing you to proc multiple enchantments on each hit.
