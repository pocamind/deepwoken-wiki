## Description

The Broodplate Helmet is an [Equipment](Equipment.md) item under the "Head" category.

Grants the user the **Broodlord's Scream** Unique equipment talent, which upon using Sing, Taunt or Ardour Scream applies anti-heal for the duration of the move.
