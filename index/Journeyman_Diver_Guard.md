# Journeyman Diver Guard

Template:EnemyInfobox1.0

## Description

Journeyman Diver Guards are NPCs that act as the guards of [Castle Light](Castle_Light.md).

They are able to use all four basic Attunements, the only NPCs in the game capable of doing so.

A Journeyman Diver Guard's appearance only consists of a [Diver Journeyman Robes](Diver_Journeyman_Robes.md) outfit with their [race](Aspects.md) varying.
