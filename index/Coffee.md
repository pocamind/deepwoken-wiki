## Description

**Coffee** is a [Food](Food.md) item. It is exclusively obtained from the "Coffee" Machine in [The Hall of Shattered Wills](The_Hall_of_Shattered_Wills.md).

It is a type of **Generic**.

### Effect

When consumed, it replenishes 2% Stomach and 30% Water.
