## Description

The **Shiva's Codex** is an unusable tome [Relic](Relics.md) that is only crafted into other [Equipment](Equipment.md).

This item is **bankable.**

## Crafting

The Shiva's Codex can craft into a few items with the [Lost Forge](Lost_Forge_of_Leiuna.md) found at the [Lost Forge of Leiuna](Lost_Forge_of_Leiuna.md).

1.  [Shiva's Pendant Earrings](<Shiva's_Pendant_Earrings.md>): [Amethyst Pendant Earrings](Amethyst_Pendant_Earrings.md) + 1x [Doom of Caeranthil's Scale](<Doom_of_Caeranthil's_Scale.md>) + 1x Shiva's Codex
2.  [Ossified Phalanx Boots](Ossified_Phalanx_Boots.md): [Phalanx Heavy Boots](Phalanx_Heavy_Boots.md) + 1x Shiva's Codex + 1x [Ethereal Osseous Plate](Ethereal_Osseous_Plate.md)
3.  [Ossified Barrel Helm](Ossified_Barrel_Helm.md): [Barrel Helm](Barrel_Helm.md) + 1x Shiva's Codex + 1x [Ethereal Osseous Plate](Ethereal_Osseous_Plate.md)
4.  [Ossified Simple Pauldrons](Ossified_Simple_Pauldrons.md): [Simple Pauldrons](Simple_Pauldrons.md) + 1x Shiva's Codex + 1x [Ethereal Osseous Plate](Ethereal_Osseous_Plate.md)
5.  [Ether Refined Mantle](Ether_Refined_Mantle.md): Sky Warlock's Mantle + 1x Shiva's Codex + 1x [Ether Refined Hide](Ether_Refined_Hide.md)
6.  [Ether Refined Brim](Ether_Refined_Brim.md): Sky Warlock's Brim + 1x Shiva's Codex + 1x [Ether Refined Hide](Ether_Refined_Hide.md)
7.  [Blizzard Ring](Blizzard_Ring.md): 1x [Frost Ring](Frost_Ring.md) + 1x [Gale Enhanced Ring](Gale_Enhanced_Ring.md) + 1x Shiva's Codex.
