# Well-kept Gravestone

| Well-kept Gravestone | |
|---|---|
| aspect | Vesperian |
| location | Isle of Vigils |

## Description

Well-kept Gravestone is an "NPC" that is part of [Suri's](Suri.md) quest, it's the gravestone of Suri's brother.

## Location

Well-kept Gravestone is located in [Isle of Vigils](Isle_of_Vigils.md) beside the barber and [Cave Master](Cave_Master.md)'s cave.
