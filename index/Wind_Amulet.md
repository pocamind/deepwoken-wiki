## Description

The Wind Amulet is a Mythic Torso [Equipment](Equipment.md) that released in the September 27th 2024 patch.

It also comes with the [Talent](Talents.md): Jetstream. Which grants you a wind armor given that you land 3 (5 for light weaponry) hits without taking any damage. The wind armor functions exactly like the one [Duke Erisia](Duke_Ishamon_Erisia.md) has.

## Trivia

-   It can be dyed, but it only affects the chain part of the amulet, so the green stone stays green.
