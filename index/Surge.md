# Surge

## Description

Template:NPC name Surge is a Canor man located in the depths, and seems to be extremely knowledgeable. He is interested in the deep owl, and will teach you tricks of the mind in return for Void Feathers.

## Location

In order to get to Surge, you must enter the [Deep Widow](Deep_Widow.md) cave in The Depths. Note that the Widow will continually pull you if you do not kill it beforehand, but it is possible to talk to Surge without killing it.

  

## Notes

-   There are no known requirements for the Visionshaper oath besides not having any other [Oaths](Oaths.md) (and a void feather)
-   You can only learn illusion magic and the Visionshaper talents if you bring Surge more void feathers after obtaining the Oath
-   The first time you give him a void feather he will teach you the Oath, not Illusion magic
