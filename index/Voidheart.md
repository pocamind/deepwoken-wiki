# Voidheart

File:VoidheartV2.png

## Description

The Voidheart is a location that is rumored to be a dev area, but, as of now, it acts like a world border, as its the edge of the sea. Sailing here can be quite difficult though, as it takes a long time to get there, as you need to sail for a long time in the Voidsea. Proof of its existence: [https://www.youtube.com/watch?v=aai\_TZimCsU](https://www.youtube.com/watch?v=aai_TZimCsU)
