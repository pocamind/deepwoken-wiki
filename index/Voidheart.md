# Voidheart

File:VoidheartV2.png

## Description

The Voidheart is a location that is rumored to be a dev area, but, as of now, it acts like a world border, as its the edge of the sea. Sailing here can be quite difficult though, as it takes a long time to get there, as you need to sail for a long time in the Voidsea. Proof of its existence: [https://www.youtube.com/watch?v=aai\_TZimCsU](https://www.youtube.com/watch?v=aai_TZimCsU). The Voidheart is currently completely empty, and not worth the time. It will most likely have something eventually but for now it is a waste of time to sail to, unless you want a cool screenshot.

## Rumors

_The World Serpent_ is a boss rumored to spawn in Voidheart. There is no evidence pointing towards this, nor any denying its existence.

Bringing a void feather to Voidheart will reveal some lore, with the Depths Trials voices talking and a talent called endless unlocked. This has been proven false by many people.
