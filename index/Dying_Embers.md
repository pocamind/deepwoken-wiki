**Dying Embers** are an item in _Deepwoken_.

## Description

Dying Embers are used to get [Flamecharm](Flamecharm.md), one of the six Attunements.

## Obtainment

Dying embers drop from any NPC that uses the [Flamecharm](Flamecharm.md) attunement.
