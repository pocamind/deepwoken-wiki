# Duelist's Attire

## Description

Template:ArmorInfobox**Duelist's Attire** is an Adept [Outfit](Outfits.md) in _Deepwoken_.

### _Requirements_

This Outfit has no cost and requires five (5) Glass Stones.

This Outfit requires [Power Level](Power.md) 15.

### _Armor Stats_

This Outfit has 1200 durability and grants +20% Physical resistance and +10% Slash resistance.
