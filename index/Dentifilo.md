# Dentifilo

Template:Food Template V1

## Description

A small mushroom with a dotted cap. It can commonly be found in [Etris](Etris.md), [Minitrysa](Minityrsa.md), and \[Placeholder\], in groups of ~3.

## Recipes used in

-   [Mushroom Soup](Mushroom_Soup.md) (Any 2 muchrooms)
-   [Mushroom Omelette](Mushroom_Omelette.md) (2 eggs, 1 mushroom)
-   Mushroom Bisque (1 of Any Mushroom, 1 Crustacean Meat)

## Trivia

-   Currently the smallest type of mushroom in game.
-   If used in [Alchemy](Alchemy.md), the primary affect applied is **Soothing**; Secondary effect is **Disgusting**.
-   One of 4 mushroom types, along with the [Gobletto](Gobletto.md), [Browncap](Browncap.md), and Bluecap.
