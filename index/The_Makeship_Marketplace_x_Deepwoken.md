# The Makeship Marketplace x Deepwoken

| The Makeship Marketplace x Deepwoken | |
|---|---|
| type | Promotional Event |

## Description

Explore and purchase physical Deepwoken Merch in the Makeship Marketplace.

The Makeship Marketplace can be accessed through the Merch and Avatar Shop section within the menu, or by [directly going to the game itself](https://www.roblox.com/games/115700747039144/The-Makeship-Marketplace-on-Roblox).

## Contents

Inside the marketplace, a large Deepwoken-themed store section features Makeship-related products that have ongoing campaigns or are permanently available.

## Trivia

-   The popup that appears when the game is accessed through the Roblox website erroneously displays the Deepwoken Wiki logo rather than Deepwoken itself.
