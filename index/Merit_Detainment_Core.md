# Merit Detainment Core

LocationL Merit Detainment Core LocationR

Deep below the fortress, all that the Authority wishes to extinguish are buried.

  

## Overview

**Merit Detainment Core** is a location in _Deepwoken_. It is complex prison in [Fort Merit](Fort_Merit.md) that you can enter without any requirements. It is controlled by [The Central Authority](The_Central_Authority.md), containing only the most dangerous people.

This dungeon is recommended for a party of people preferably [Power](Power.md) 20.

## Trivia/Notes

-   After disabling the security system, your taunt gets replaced with a cannon whistle used by the guards. This allows the player to use the artillery to shoot cannonballs at their cursor.
