# Whaler Initiate

Template:ArmorInfobox**Whaler Initiate** is a Starter [Outfit](Outfits.md) in _Deepwoken_.

## Statistics

This Outfit requires [Power Level](Power.md) 5, however, it appears to only be obtainable by spawning with the [Voidwalker Origin](https://deepwoken.fandom.com/wiki/Origins#Voidwalker) selected.

### _Armor Stats_

This Outfit has 1500 durability and grants +15% Physical resistance, +5% Elemental resistance and +25% Stealth.

## Trivia/Notes

-   This Outfit is granted for free upon starting with the [Voidwalker Origin](https://deepwoken.fandom.com/wiki/Origins#Voidwalker), however is lost once crafting/equipping another armor.
-   This Outfit appearance is the old version of the [Whaler Coat](Whaler_Coat.md) Outfit.
