# Light Eater Ring

Light Eater Ring is an [Equipment](Equipment.md) item under the "Rings" category.

## Description

Light Eater Ring grants the user +2% Fire Armor, +2% Lightning Armor, and one Uncommon Pip upon being equipped. It also causes the user to regenerate ether upon taking damage from [Flamecharm](Flamecharm.md) mantras, or [Thundercall](Thundercall.md) mantras, or burn. The ether regeneration scales on damage taken. The user must be at Power 3 to equip this ring.
