# Crypt Blade

File:Crypt Blade.png

## Description

The Crypt Blade is a Legendary weapon only obtainable in the depths. It scales with [Shadowcast](Shadowcast.md) (10) instead of heavy weapon with 26 base damage.

## Abilities

\- It procs every shadowcast passive/talent.

\- Highest damaging weapon in the game
