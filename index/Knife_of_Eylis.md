# Knife of Eylis

Template:Creature Info Box v2

## Description

A Whaler is a type of enemy NPC that wields either a [Whaling Knife](Whaling_Knife.md) and can be found in [The Aratel Sea](The_Aratel_Sea.md) fighting Etrean Sailors on 2 burning [Frigates](Ships_&_Sailing.md). 2 Whalers will spawn accompanying a Whaler Captain per event.
