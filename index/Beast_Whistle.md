# Beast Whistle

| Beast Whistle | |
|---|---|
| type | Talent |
| bankable | Yes |
| droppable | Yes |
| pass down | No |
| desc | Call forth your mount |

## Description

The Beast Whistle is an item granted upon completing [Sordino's](Sordino.md) quest. It allows you to summon your mounts virtually anywhere provided there is solid ground. This includes but is not limited to the Overworld, Depths, and even Dungeons (though this is not very practical as the sole purpose of dungeons is to fight off enemies which will inevitably put you in combat, disabling the mount).

## Trivia

-   If you will try to use it during combat, you will get next notification "You lack the language to command this beast in battle."
    -   This could possibly be a hint that there very well could be a vow/oath that allows you to command beasts in battle.
-   While mounted on a beast, you can completely resist the slow effect of the wind in Layer 2, but still can be pushed away.
-   Beast walks with speed of 22 stud/sec and runs with speed of 34.5 stud/s
-   Equipping a weapon, using a mantra, or getting combat tagged instantly dismounts you from tamed beast.
-   Having a mount doesn't negate fall damage but slightly reduces it by around 2%. This value needs further testing.
-   The whistle can bug out and won't work if you trying to use it while riding a thresher. Rejoin to fix
