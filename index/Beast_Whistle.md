# Beast Whistle

## Description

The Beast Whistle is an item granted upon completing [Sordino](Sordino.md)'s quest. It allows you to summon your [mounts](Mounts.md) virtually anywhere provided there is solid ground. This includes but is not limited to the Overworld, Depths, and even Dungeons, so long as you are not in combat. Trying to use the whistle in combat will display the message: "You lack the language to command this beast in battle."

The whistle can bug out and won't work if you trying to use it while riding a thresher. Rejoin to fix

## Trivia

-   The combat tag message could possibly be a hint that there could be a vow/oath that allows you to command beasts in battle.
