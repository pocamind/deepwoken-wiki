# Beast Whistle

| Beast Whistle | |
|---|---|
| type | Talent |
| bankable | Yes |
| droppable | Yes |
| pass down | No |
| desc | Call forth your mount |

## Description

The Beast Whistle is an item granted upon completing [Sordino's](Sordino.md) quest. It allows you to summon your mounts virtually anywhere provided there is solid ground. This includes but is not limited to the Overworld, Depths, and even Dungeons.

## Trivia

-   If you will try to use it during combat, you will get next notification "You lack the language to command this beast in battle."
    -   This could possibly be a hint that there very well could be a vow/oath that allows you to command beasts in battle.
-   While mounted on a beast, you can completely resist the effects of the wind in Layer 2.
-   Equipping a weapon, using a mantra, or getting combat tagged instantly dismounts you from tamed beast.
-   Having a mount doesn't negate fall damage. It could potentially reduce it but more testing is needed.
