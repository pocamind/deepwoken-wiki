# Vigil Swordsman

Template:Creature Info Box v2

## Description

An Etrean Footsoldier is type of enemy NPC that can be found in [Erisia](Erisia.md) and the [Starswept Valley](The_Starswept_Valley.md). Unlike the [Etrean Guardsmen](Etrean_Guardsmen.md), the Etrean Footsoldier's clad is colored white, they can be found fighting Authority Footsoldiers in [Erisia](Erisia.md), or Hive Footsoldiers if found in the [Starswept Valley](The_Starswept_Valley.md); a battle going both ways. They are usually equipped with a [Sword](<Sword_(weapon).md>), [Mace](Mace.md) or [Spear](Iron_Spear.md) along with the [Galebreathe](Galebreathe.md) Attunement; they also spawn randomly just like how any event would spawn.
