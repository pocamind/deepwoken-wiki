# JOE's

| JOE's | |
|---|---|
| bankable | Yes |
| droppable | Yes |
| pass down | No |
| desc | It stands for Judicator's Original Enchantment. An enchant that was supposed to be only for the strongest of monster slayers. No one knows what happened to the original Judicator who made this enchant. |
| obtainment | Win certain events held by the developer Agamatsu |

## Description

JOE's is an **unobtainable** enchantment with unknown properties.

The visual change for your weapon is unknown, as nobody has this enchantment, though its [Enchant Stone](Enchant_Stone.md) glows red, similar to [Curse of the Bloodthirsty](Curse_of_the_Bloodthirsty.md)'s Enchant Stone.

## Effect

JOE's's effects are entirely unknown, though based on its description, it might be a PvE-based enchantment.

## Trivia

-   No non-developer player has ever obtained this enchantment, as every event that occurred with this as the reward have ended with no winners. This is largely why this enchantment's effects are undocumented.
