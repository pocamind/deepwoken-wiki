# JOE's

## Description

JOE's is an **unobtainable** enchantment with unknown properties.

This enchantment changes your weapon's color to a glowing red with floating red vertical particles emanating from it.

## Effect

JOE's's effects are entirely unknown, though based on its description, it might be a PvE-based enchantment.

## Trivia

-   Allegedly, the effects of this Enchantment are not in the game currently.
