# Reservoir

This is what the Reservoir bar looks like

  
This bar controls the speed ether regenerates, when the bar reaches 0 you regen your ether significantly slower, your reservoir depletes when ether is being regenerated. The bar will also not regenerate unless ether is full. The only exception to this rule is sitting next to a campfire which will regen both your ether and reservoir.

**(Note: This mechanic is set to be removed in the same update that Layer 2 of the Depths is released)**
