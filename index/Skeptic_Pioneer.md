# Skeptic Pioneer

## Description

Template:ArmorInfobox **Skeptic Pioneer** is a Starter [Outfit](Outfits.md) in _Deepwoken_.

### _Requirements_

This Outfit costs and requires five (5) pieces of Cloth and Fiber and one (1) Crystal Lens to craft.

### _Armor Stats_

This Outfit has 1200 durability and grants the user with +8% Physical resistance and +8% Elemental resistance.

This Outfit also grants the [Talent Card](Talents.md) _Apothecary_, which makes potions that benefit the player more potent.

## Location

This Outfit's Schematic can be found:

-   It can be found in chests in [Lower Erisia](Lower_Erisia.md).
-   It _might_ be able to be purchased above the weapon shop in Etris.
