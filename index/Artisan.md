# Artisan

| Artisan | |
|---|---|
| location | Guild Base |

## Description

The Artisan is an NPC found in the [Guild Base](Guild_Bases.md) who is capable of [smelting](Transmogrification.md) items for 800. They have the same function [Jarvis](Jarvis.md).
