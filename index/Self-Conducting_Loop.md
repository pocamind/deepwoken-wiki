# Self-Conducting Loop

Self-Conducting Loop in the Mantra shop at Etris.

## **Description**

_"A ball of revolving, electric volts, as if it could power a town."_

Used to train the [Thundercall](Thundercall.md) attunement. It can be used 200 times. Upon activation, the character rubs their hands, conducting a static current.

_Purchased at any Mantra Shop -_ _**25 [Notes](Notes.md)**_

## Trivia

-   Every attunement has a training item.
-   Can break if overused.
