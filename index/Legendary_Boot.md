# Legendary Boot

| Legendary Boot | |
|---|---|
| name | Legendary Boot |
| location | Erisia |

## Descrition

The legendary Boot is an NPC that searches for someone worthy to wear it. It is possible to be worthy by having the [Worthy Regalia](Regalia.md).

## Location

The Legendary Boot is located in [Erisia](Erisia.md) on top of a rock, that is in an opening after passing through a waterfall.
