**Memory Shards** were a [Relic](Relics.md) given to you when you spawn in the [Vow of Iron](Vow_of_Iron.md) gamemode to restart the fight against the mode's mandatory [Bosses](Bosses.md).

## Functionality

When you would reach 0% HP in a memory fight, you would be given a 75% HP Heal and the boss would fully heal, along with setting your [Etris Flasks](Etris_Flask.md) to 1

## Trivia

-   On release, you had to manually pull out and use a Memory Shard in order to restart the fight against a Memory Boss when you were knocked, and you could only use it if you had been combat-tagged by them.
    -   Some days after the release of Vow of Iron however, this was changed and they are now automatically used whenever you would get knocked, even if you don't have any in your inventory.
-   The description previously read "Use these as you're fighting a Memory Boss to restart the fight when you're knocked. For each use of these however, gain increasing amounts of Wither when the fight is restarted." However, as these became a passive instead of needing to be used, they became useless and only used to gain 600 notes at the start of the playthrough. Thus they were removed from starting obtainment and any that you had prior could just be sold, similar to Dye Removers.
