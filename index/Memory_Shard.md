**Memory Shards** were a [Relic](Relics.md) given to you when you spawned in the [Vow of Iron](Vow_of_Iron.md) gamemode to restart the fight against the mode's mandatory [Bosses](Bosses.md).

## Usage

When you would get knocked in a Memory Boss fight, you would be given 25% [Wither](Status_Effects.md#wither) and a full heal up to 75% and the Memory Boss would fully heal as well, along with setting your [Etris Flasks](Etris_Flask.md) to 1 and doubling its healing percentage.

## Trivia

-   On release, you had to manually pull out and use a Memory Shard in order to restart the fight against a Memory Boss when you were knocked, and you could only use it if you had been combat-tagged by them.
    -   Some days after the release of Vow of Iron however, this was changed and they were automatically used whenever you would get knocked, even if you didn't have any in your inventory.
        -   Memory Shards themselves were removed from being given out to [Vow of Iron characters](Mementos.md) on spawn shortly after this change, and their description was changed to _"Useless"_, but they were not removed from existing characters' inventories. Their description previously read _"Use these as you're fighting a Memory Boss to restart the fight when you're knocked. For each use of these however, gain increasing amounts of Wither when the fight is restarted."_
