## Description

A Lionfish Scale can be obtained from [Lionfishes](Lionfish.md) which can be found in [The Depths](The_Depths.md), [The Void Sea](Voidsea.md), and the [Eastern Luminant](Maps.md), it can be used to craft the [Ignition Union Uniform](Ignition_Union_Uniform.md), [Pathfinder Arch-Sorcerer](Pathfinder_Arch-Sorcerer.md), [Authority Captain Uniform](Authority_Captain_Uniform.md) and the [Whaler Coat](Whaler_Coat.md).
