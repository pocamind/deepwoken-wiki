# Ring of Wisps

**Ring of Wisps** is an [Equipment](Equipment.md) item in _Deepwoken_.

## Description

Ring of Wisps grants the user one Uncommon Pip upon being equipped. It also grants the user double the amount of charges of Attunement Wisps.
