# Prophet's Ring

Prophet's Ring is an [Equipment](Equipment.md) item under the "Rings" category.

## Description

Prophet's Ring grants the user one Legendary Pip upon being equipped. It also grants every weapon 1.2 charisma scaling. The user must be at Power 6 to equip this ring.
