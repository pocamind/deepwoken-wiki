# Heratic's Sutra

Heratics Sutra is an Ability that can be obtained in Layer 2 of The Depths.

File:Heratic's Sutra Talent Card.png

**_Card for Heratic's Sutra_**

**Obtainment:**

It's obtainment involves 80 Willpower and Tier 1 Insanity, by which you would then head to Layer 2 by either void or by Klaris, and you would then locate the NPC Kasen. You can watch this video here for a tutorial on how to get it:

[https://www.youtube.com/watch?v=uVUnkoyqOHo](https://www.youtube.com/watch?v=uVUnkoyqOHo)

**Abilities:**

The use of Heratic's Sutra causes you to gain insanity, which can be useful if you are using talents such as Lose your Mind, in which Heratic's Sutra can be crucial for that +15% damage buff. It is advised that you also get Unfazed with this, as it would prevent the damaging side effects of insanity.
