# Strange Tomes

Volume #2

File:Tome description.png

Volume #2 Description

A strange tome that seems to be orginated from the ministry with each volume having different origin, combining the 3 tomes in the basement of [Outpost Antumbra](Outpost_Antumbra.md) will give the player [Bloodfouler](Bloodfouler.md)

Volume 1 Obtainment: [Volu](Volu.md) NPC near [Trial of One](Trial_of_One.md), [Outpost Antumbra](Outpost_Antumbra.md)

Volume 2 Obtainment: [Chaser, Scholar of the Crimson Contract](Chaser,_Scholar_of_the_Crimson_Contract.md) (Not guaranteed but high chance to drop).

Volume 3 Obtainment: [The Understrand](The_Understrand.md) (Entered not via [Silhuett](Silhuett.md) as this is a separate event which does not spawn a chest, note that this tome is also not guaranteed).
