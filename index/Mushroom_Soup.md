# Mushroom Soup

## Description

Template:Food Template V1Mushroom Soup is a food item present in the game. More info will be added to the page when Deepwoken is released.

## Recipe

Mushroom Soup is crafted with any combo of two different mushrooms at a campfire.

**Example:** _Dentifillo + Gobletto **OR** Dentifillo + Browncap **OR** Browncap + Gobletto_
