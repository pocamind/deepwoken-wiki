# Ministry Agent

Template:EnemyInfobox1.0

## Description

The Ministry Cache Agent is an NPC located within [Scyphozia, The First Layer](First_Layer.md). He can be found by using Shadow Travel / Shadow's Flight near Silhuett, who spawns randomly around the first layer.

## Moves

The Agent wields the Deepspindle along with a variety of [Shadowcast](Shadowcast.md) mantras.
