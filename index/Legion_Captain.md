# Legion Captain

## Description

Legion Captain is NPC Type enemy that is found in Hive and the camp in Vipers Jaw. Typically after defeating them you will get bad rep with the legion preventing you getting the captains recommendation.

Legion Captain are found all around Hive in different locations usually fighting the Authority Captain.

### Moves

-   **Different Weapons:** Fast Normal M1 hits with Swords, Spears and Fists.

-   **The Captain uses moves from both Galebreath and Attunement-less:** Rising Wind, Heavenly Wind, Gale Lunge, Strong Left, and Rapid Punches.
-   There is another NPC that uses Grand Javelin with Grand Skewer, **this is not the captain.**

### Tips

-   They usually are defeated in the Hive event so you can use the Authority Captain to knock him if you need.
