# Mantra Modifiers

This page is a list of all items that you can use in order to modify a mantra (add modifying items and how they modify a mantra).

  

## Mantra modifiers

\- Cloudstone- increases mantra size

\-Driftwood- increases casting speed

\-Crystal Lens- Increases mantra range

\-Magnet Spark- makes projectile spells homing
