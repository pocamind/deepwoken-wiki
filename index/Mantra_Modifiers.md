# Mantra Modifiers

This page is a list of all items that you can use in order to modify a mantra (add modifying items and how they modify a mantra).

-   Gale stone+Jolt grab= Jolt grab went from a short dash and grab=slam down animation to a charge up to teleport with same effect
-   Burning Servants + Cloudstone = Increases amount of servants
