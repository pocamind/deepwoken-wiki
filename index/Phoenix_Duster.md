# Phoenix Duster

Phoenix Duster (dyed red) being worn by Zartania.

The Phoenix Duster is a [developer-exclusive](Developer_Specs.md) item.

One of the rarest items in the game due to this. The stats are unknown, and because it is an admin item, it may be extremely overpowered compared to normal items. Also, because it is a phoenix, we can speculate it has similar effects to the [Immortal Helm](Immortal_Helm.md), meaning it can allow you to cheat death, but this is unconfirmed.
